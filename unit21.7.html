Describe which are the appropriate filetypes for websites

There are a lot of different filetypes used on the internet, and websites make use of a lot of them, whether it’s to render the page being viewed, or to link to an asset or source.

The key examples for generating a website page are .html or .htm which act much like a plain text file, but the content tells the browser how to load it. In fact if a web server isn’t configured correctly it will display all of the text (source) in the browser.

Then you have .CSS which gives a deeper level of control over the styling of the page. This can be useful for causing an effect on multiple elements with a single command, which helps ensure the page can render quickly.
